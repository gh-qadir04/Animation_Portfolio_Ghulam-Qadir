## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/gh-qadir04/Animation_Portfolio_Ghulam-Qadir.git

# Navigate to project
cd Animation_Portfolio_Ghulam-Qadir

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

## 🔍 SEO Implementation

This portfolio includes comprehensive SEO optimization:

- ✅ **Meta Tags** — Title, description, keywords, robots
- ✅ **Open Graph** — Facebook/LinkedIn sharing cards
- ✅ **Twitter Cards** — Twitter/X sharing cards
- ✅ **Schema.org JSON-LD** — Person, WebSite, Projects, Breadcrumb structured data
- ✅ **Canonical URL** — Prevents duplicate content issues
- ✅ **Sitemap.xml** — 6 URLs with priorities and change frequencies
- ✅ **Robots.txt** — Allows all crawlers including AI bots (GPTBot, ClaudeBot, PerplexityBot)
- ✅ **Semantic HTML** — `<header>`, `<section>`, `<article>`, `<footer>`
- ✅ **Alt Text** — Descriptive alt text for all images
- ✅ **Favicons** — Multiple sizes for all devices
- ✅ **Security Headers** — X-Frame-Options, X-Content-Type-Options, Referrer-Policy
